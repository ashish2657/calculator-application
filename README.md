# calculator-application
Android Calculator App with Splash Screen
Project Description

This project is a simple Android Calculator Application developed using Android Studio. The application performs basic arithmetic operations such as Addition, Subtraction, Multiplication, and Division.

The app also includes a Splash Screen that appears when the application starts before opening the calculator interface.

This project is created for learning Android Programming and understanding how to design UI using XML and implement functionality using Java.

Features

Splash Screen when the app starts

Simple and user-friendly calculator interface

Perform basic arithmetic operations

Addition (+)

Subtraction (-)

Multiplication (*)

Division (/)

Clear button to reset input

Two-line calculator display design

Technologies Used

Android Studio

Java (Backend Programming)

XML (UI Design)

Project Structure
1. Splash Screen

Displays the app logo or welcome screen for a few seconds before opening the calculator.

2. Main Calculator Screen

The main screen contains:

Display area for numbers

Buttons for digits (0–9)

Arithmetic operation buttons

Equal button to show result

Clear button

Files Used in Project
1. Activity_main.xml

Used to design the calculator user interface

Contains buttons and text display

2. MainActivity.java

Handles calculator logic

Performs arithmetic calculations

Manages button click events

3. SplashActivity.java

Controls the splash screen timing

Opens the main calculator activity

4. AndroidManifest.xml

Declares application activities

Sets SplashActivity as the launcher activity

5. colors.xml

Defines colors used in the application

6. themes.xml

Defines the application theme and style

How to Run the Project

Open Android Studio

Click Open Existing Project

Select the project folder

Build the project

Run the app on Android Emulator or Physical Device

Learning Objectives

Understanding Android Activity

Designing UI using XML Layout

Implementing Java logic in Android

Creating a Splash Screen

Handling button click events
